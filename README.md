# Vue 3 + Vite
后台管理系统基础框架，技术栈：vue3 + vite + pina + vue-router + vue-request + axios + ant-design  
vue: 3.2.25  
node: v16.17.0  
npm: 8.15.0  


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
